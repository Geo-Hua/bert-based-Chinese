This document is the reproducibility instruction for “Revealing the co-occurrence patterns of the group emotions from social media data”.

This document provides a detailed explanation of how to reproduce all experimental results, figures and tables presented in the paper, and the key indicators in the abstract by using the shared datasets and source code. The aim is to ensure full reproducibility of the study.

Directory Structure after Extraction or Cloning:
├── code/ # Scripts for data processing and modelling
├── data/ # Raw and intermediate datasets
├── results/ # Output results
├── README.docx # instructions
├── README.md # instructions
└── requirement.txt # requirement instructions
Citation for data source:
Liu, Z. et al. (2024). Shifting sentiments: analyzing public reaction to COVID-19 containment policies in Wuhan and Shanghai through Weibo data. Humanities and Social Sciences Communications, 11, 1104.

Notes and Recommendations:
Path Handling: All scripts use relative paths; no modification is necessary.
Environment Check: Ensure the Python environment is correctly configured to avoid dependency issues.
First-Time Use: It is recommended to run and debug each script step-by-step to better understand the logic.
Code Annotations: All scripts are well-commented to facilitate review and modification.
